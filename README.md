# kb
kb is a tool to easily save and find your snippets or knowledge base articles.


kb adds to and finds snippets from /home/$USER/kb. For easy use,
add e.g. alias "kb" to run "/path/to/your/python/ kb.py".


For usage help, run 
```
    python3 kb.py --help
```
